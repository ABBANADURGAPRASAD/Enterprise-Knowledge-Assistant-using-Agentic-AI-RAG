# Assets

Project assets including architecture diagrams, logos, and screenshots.

## Contents

```
assets/
├── diagrams/     # Architecture and flow diagrams (PNG/SVG)
├── logos/        # Application logos and branding
└── screenshots/  # UI screenshots for documentation
```

## Usage

Reference assets in documentation using relative paths:
```markdown
![Architecture](../assets/diagrams/system-overview.png)
```
